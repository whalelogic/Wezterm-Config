#### Features

- Borders
- Fancy Tab Bar
- Status Line
- Git-enabled
- Nerd Fonts
- Zsh 5.9

#### Basic Commands 

- Ctrl+b (prepatory command)
- Ctrl+b then h (execute 'h') (Split window Horizontally)
- Ctrl+b then v (execute 'h') (Split window Vertically)
- Ctrl+Shift+ArrowKey(Up,Down,Left,Right) (Switch window Pane)
